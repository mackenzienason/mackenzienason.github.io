Atomic design

Odds‚Äô Design System was built using Brad frost‚Äôs Atomic Design. Atomic design is a modular approach to constructing interfaces.

Fundamentally it draws from chemistry and dissects an interface into 5 main parts:

  • atoms (styles)
  • molecules
  • organisms
  • templates
  • pages

This break down is how we built our system. Starting at the atom level, then building organisms, using organisms to create molecules, molecules to create organisms... etc.

Outcome

The result of the project was a system that dramatically increased productivity, stakeholder transparency, onboarding and gave the team more time to rapidly iterate ideas into high fidelity prototypes.
